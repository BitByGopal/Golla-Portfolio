# 🖥️ Golla Gopal — macOS Portfolio

A fully interactive **macOS-style portfolio** built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies — just clean code that looks and feels like a real operating system.

🌐 **Live:** [BitByGopal.github.io/portfolio](https://BitByGopal.github.io/portfolio)

---

## ✨ Features

### 🔐 Boot & Lock Screen
- Terminal typewriter boot animation
- Split lock screen with sliding info cards (origin story, hobbies, projects, stats)
- Rotating developer quotes
- Day/Night auto-detect with manual toggle

### 🖥️ Desktop
- macOS menubar with live weather (Phagwara, Punjab)
- Mini calendar widget with month navigation
- Right-click context menu
- Day ☀️ / Night 🌙 theme toggle

### 📁 Apps

| App | Description |
|-----|-------------|
| **Terminal.app** | Interactive terminal with 15+ commands, Tab autocomplete, command history |
| **Portfolio.app** | Finder-style app with sidebar navigation — About, Skills, Projects, Internship, Education, Certificates, Contact |
| **Classic.app** | Premium scrollable single-page portfolio with editorial design |
| **Games.app** | 4 games — Snake (skills), Flappy Bird (projects), 2048 (certificates), Memory Cards (projects) |
| **Assistant.app** | AI-style rule-based chatbot that answers questions about Gopal |
| **Book a Call** | EmailJS-powered scheduling form — sends directly to Gmail |

### 🎮 Games
- 🐍 **Snake** — Reveals tech skills as you score
- 🐦 **Flappy Bird** — Reveals projects with full descriptions
- 🔢 **2048** — Reveals all 12+ certificates & achievements
- 🃏 **Memory Cards** — Flip cards to reveal project screenshots
